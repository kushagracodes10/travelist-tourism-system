# Travelist Tourism Management System

A Java Swing-based tourism application that allows users to explore popular Indian travel destinations through an interactive graphical interface.

## Features

* Interactive destination selection
* Detailed city information
* Tourist attraction guides
* Weather information integration
* Transportation information
* Modern Java Swing GUI
* Multiple destination pages

## Destinations Covered

* Jaipur
* Goa
* Varanasi
* Darjeeling

## Technologies Used

* Java
* Java Swing
* AWT
* JSON API Integration

## How to Run

1. Open the project in IntelliJ IDEA.
2. Ensure all project files are present.
3. Run `frontpage.java`.
4. The Travelist application will launch.

## Project Overview

Travelist is a desktop-based tourism management application developed as a college project. The application provides destination information, local attractions, transportation details, and weather-related information through an interactive user interface.
